# 📊 Data Pipeline com Dashboard Interativo

Projeto end-to-end de dados com foco em engenharia, análise e visualização.

## 🚀 Funcionalidades

- Coleta de dados via API
- Tratamento e transformação com Pandas
- Geração de dados sintéticos
- Armazenamento em SQLite
- Dashboard interativo com Streamlit
- Filtros dinâmicos e KPIs

## 🛠️ Tecnologias

- Python
- Pandas
- SQLAlchemy
- Streamlit
